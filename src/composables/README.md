# Composables

## Purpose
Vue 3 composables for reusable logic shared across multiple components and modules.

## Planned Composables
- `useDragAndDrop.js` (Phase 3) - Handle drag and drop logic for players and ball.
- `useKeyframes.js` (Phase 5) - Manage keyframe creation and navigation.
- `useAnimation.js` (Phase 6) - Control playback and interpolation.
- `useField.js` (Phase 2) - Field calculations and coordinate conversions.

## Usage Notes
- Keep composables focused on one concern.
- Expose refs/computed values and explicit action functions.
