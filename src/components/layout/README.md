# Layout Components

## Purpose
Contains the main application shell components that define the overall page structure for the Tactical Board Rugby interface.

## Planned Components
- `AppHeader.vue` (Phase 1, Task 1.5) - Top navigation and global actions.
- `LeftSidebar.vue` (Phase 1, Task 1.5) - Tools, player controls, and creation options.
- `RightSidebar.vue` (Phase 1, Task 1.5) - Properties, inspector, and contextual options.
- `TimelineFooter.vue` (Phase 1, Task 1.5) - Timeline and playback area at the bottom.

## Usage Notes
- These components are currently skeleton placeholders.
- Layout composition will be wired into views once Phase 1 layout tasks begin.
