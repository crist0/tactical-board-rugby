# Field Components

## Purpose
Hosts components responsible for rendering the rugby field, zones, and visual field markings.

## Planned Components
- `RugbyField.vue` (Phase 2) - Main field rendering surface.
- `FieldLines.vue` (Phase 2) - Goal lines, halfway line, and other field markings.
- `GridOverlay.vue` (Phase 2) - Optional alignment/grid helper overlay.

## Usage Notes
- Coordinate systems and scaling helpers will be shared with `src/composables/useField.js`.
