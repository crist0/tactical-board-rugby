# Player Components

## Purpose
Contains player-related visual and interactive components used to place and manage team members on the field.

## Planned Components
- `PlayerToken.vue` (Phase 3) - Draggable player marker.
- `PlayerLabel.vue` (Phase 3) - Player number/name label presentation.
- `PlayerGroup.vue` (Phase 4) - Team-level grouping and batch operations.

## Usage Notes
- Drag-and-drop interactions will be powered by `useDragAndDrop.js`.
