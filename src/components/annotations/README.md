# Annotation Components

## Purpose
Contains drawing and annotation components used to explain tactics and movement on top of plays.

## Planned Components
- `ArrowAnnotation.vue` (Phase 4) - Directional movement arrows.
- `LineAnnotation.vue` (Phase 4) - Straight and curved tactical lines.
- `TextAnnotation.vue` (Phase 4) - Labels and strategic notes.
- `MarkerAnnotation.vue` (Phase 4) - Point markers and highlights.

## Usage Notes
- Annotation persistence and visibility toggles will be managed by play and UI stores.
