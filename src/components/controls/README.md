# Controls Components

## Purpose
Contains UI controls for playback, zoom, editing tools, and interaction shortcuts.

## Planned Components
- `PlaybackControls.vue` (Phase 6) - Play, pause, speed, and timeline controls.
- `ZoomControls.vue` (Phase 2) - Field zoom and viewport actions.
- `ToolPicker.vue` (Phase 4) - Select active drawing/editing tools.

## Usage Notes
- Most control state will be centralized in `useUIStore.js` and `usePlaybackStore.js`.
